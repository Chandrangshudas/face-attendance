# Face Recognition Attendance System

A **real-time face recognition attendance system** built with **Python, OpenCV, and Flask**.  
Logs attendance automatically when a known face is detected.

## Features
- Detect and recognize faces in real-time
- Automatic attendance logging in CSV
- Web-based live video feed
- Add known faces by putting images in `faces/` folder

## Tech Stack
- Python
- OpenCV
- face_recognition library
- Flask (Web Interface)
- Pandas (CSV attendance logging)
